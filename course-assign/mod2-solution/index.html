Web Development

HTML

HTML (HyperText Markup Language) provides the structure for the webpage content. It's what tells this page to have a heading, three sections with content, each one with a section title.

CSS

CSS (Cascading StyleSheet) provides the styling, making it visually appealing. It's what allows us to have three grey boxes for the section information and three individually colored ones for the section titles.

JS

JS (JavaScript) deals with behaviour, functionality and interactivity, for example: what happens when you click something.
